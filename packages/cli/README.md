# @Fernando-Ligabue/cli

> CLI for AlpineUI - Add beautiful components to your Alpine.js project

## Installation

```bash
npm install -g @Fernando-Ligabue/cli
```

## Usage

```bash
# Initialize AlpineUI in your project
alpineui init

# Add components
alpineui add button
alpineui add button dialog tabs

# Add all components
alpineui add --all

# List available components
alpineui list

# Upgrade components
alpineui upgrade
```

## Alias

You can also use the shorter alias:

```bash
au add button
au list
```
