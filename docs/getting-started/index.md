# Introduction

AlpineUI is a collection of **re-usable components** built with [Alpine.js](https://alpinejs.dev) and [Tailwind CSS](https://tailwindcss.com).

Inspired by [ChadCN UI](https://chadcn.com), AlpineUI provides beautiful, accessible components that you can copy and paste into your projects.

## Features

- **Copy & Paste** - Just copy the component code and paste it into your project
- **Lightweight** - Built on Alpine.js, the JavaScript framework that prioritizes size and performance
- **Customizable** - Each component is styled with Tailwind CSS - modify directly to match your design
- **TypeScript Ready** - Full TypeScript support with type definitions
- **Dark Mode** - All components support dark mode out of the box
- **Package Manager** - Install individual components via npm or use our CLI tool

## Why AlpineUI?

Alpine.js is a rugged, minimal tool for composing behavior directly in your markup. It's perfect for adding interactivity to simple pages or building complex applications.

AlpineUI extends Alpine.js with beautifully designed components that follow accessibility best practices.

## Quick Start

Start by installing Alpine.js and Tailwind CSS in your project, then copy the components you need.

See the [Installation](/getting-started/installation) guide for detailed instructions.
