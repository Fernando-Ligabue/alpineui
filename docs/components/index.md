## Usage

Each component can be used by:

1. **Copy & Paste** - Copy the component code directly into your project
2. **NPM Install** - Install individual packages via npm
3. **CLI** - Use the AlpineUI CLI to add components

See the [Installation](/getting-started/installation) guide for more details.
